# Processed Data

This directory contains preprocessed datasets ready for model training.

## Files

### CSV Files
- **`train_processed.csv`** - Training set (43,000+ samples)
- **`val_processed.csv`** - Validation set (5,400+ samples)
- **`test_processed.csv`** - Test set (5,400+ samples)

### Mapping File
- **`emotion_mapping.json`** - Label encoding for emotions

## Preprocessing Steps Applied

1. **Lowercase conversion** - All text converted to lowercase
2. **URL removal** - Removed http/https links
3. **HTML tag removal** - Cleaned HTML markup
4. **Email removal** - Removed email addresses
5. **Mention removal** - Removed @username mentions
6. **Hashtag normalization** - Removed # but kept the word
7. **Punctuation normalization** - Multiple punctuation reduced to single
8. **Whitespace normalization** - Extra spaces removed
9. **Short text removal** - Removed texts < 3 characters

## Data Format

Each CSV file has 2 columns:
- `text` - Preprocessed text ready for modeling
- `emotion` - Emotion label (anger, disgust, fear, joy, neutral, sadness, surprise)
